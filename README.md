# Laravel Lighthouse Demo

This is my "hello world" project for combining *Laravel + Lighthouse (GraphQL)*.


## Installation

```bash
npm install
npm run dev
```

```bash
composer install
php artisan serve
```

Update `.env`:
```dotenv
GRAPHIQL_ENABLED=true
```

Visit http://127.0.0.1:8000/graphiql


## Testing

```bash
php artisan test
```
